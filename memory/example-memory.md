This is an example memory. Each memory is one file holding one durable fact, with the
frontmatter above. `type` is one of: `user` (who you are, your preferences),
`feedback` (how Claude should work — include the why), `project` (ongoing work or
constraints not in the code), or `reference` (pointers to dashboards, tickets, URLs).

Link related memories with `[[their-name]]`. Add a one-line pointer in `MEMORY.md`
whenever you create a memory here. Delete this file once you've got the idea.
