# Provenance

This starter is the **person-agnostic, IDE-agnostic template** form of a personal
cross-IDE agent configuration maintained upstream. Changes flow **one way** — from the
maintainer's private config into this template, genericized: no personal or brand names,
first-person "I/me" voice, generic system references, and no private skill paths.

The instruction spine lives in `AGENTS.md` (the cross-tool [AGENTS.md](https://agents.md/)
standard, read by Claude Code, Codex, Cursor, and Antigravity); `CLAUDE.md` is a symlink
to it. Upstream, the same spine is synced into each tool's global instruction file.

If you forked this: it's yours. Edit freely — there is no sync back upstream.

## Portable spine

These Working Style sections are the "portable spine" — operating standards designed
to be stable across projects rather than personal preferences. They are kept aligned
with the upstream config; the rest of `AGENTS.md` (your profile, brand/voice SSOT) is
yours to define.

- Skill & Command Leverage Protocol
- Qualifying Questions Protocol
- Engineering Mantra (Security, Stability, Reliability, Compliance)
- Proportional Rigor
- Attribution & Versioned Outputs
- Third-Party Component Trust
- Data Handling
- Truthfulness & Escalation
- Look Around the Corner
- Skill-Worthiness Radar
- Next Steps & Sequenced Building
